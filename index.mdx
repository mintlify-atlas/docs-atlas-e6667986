Crowbar is an open-source Windows application designed for GoldSource and Source engine modders. It bundles everything you need to work with game models and packages — from decompiling `.mdl` files into editable source files, to compiling QC scripts, extracting VPK archives, and publishing to the Steam Workshop.

## Getting Started

<Steps>
  <Step title="Install Crowbar">
    Download the latest release from the [Steam community page](https://steamcommunity.com/groups/CrowbarTool) and extract it to a folder on your PC. No installer required.
  </Step>
  <Step title="Set up a game profile">
    Open the **Set Up Games** tab and configure a game entry with paths to your game's `gameinfo.txt`, StudioMDL compiler, and HLMV viewer. See [Game Setup](/setup/game-setup).
  </Step>
  <Step title="Decompile a model">
    Switch to the **Decompile** tab, browse to a `.mdl` file, choose an output folder, and click **Decompile**. Crowbar extracts the QC file, SMD meshes, and animations.
  </Step>
  <Step title="Edit and recompile">
    Modify the decompiled source files in your preferred editor, then use the **Compile** tab to run StudioMDL and produce a new `.mdl`.
  </Step>
</Steps>

## Supported Engines and Versions

Crowbar supports a wide range of MDL versions across multiple engines:

| Engine | Example Games | MDL Versions |
|---|---|---|
| GoldSource | Half-Life, CS 1.6, TFC | v06, v10, v2531 |
| Source | HL2, TF2, L4D2, CS:GO | v44–v53 |
| Legacy Source | Portal, EP1/EP2 | v27–v49 |

See the full [Supported MDL Versions](/reference/supported-mdl-versions) reference for the complete list.
